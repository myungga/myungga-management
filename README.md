# Myungga Korean Restaurant — Management System

Sales · Expense · Statistics Management for Myungga Restaurant, Chiang Mai

## Files
- `app.py` — Flask backend server
- `requirements.txt` — Python packages
- `render.yaml` — Render.com deployment config
- `templates/index.html` — Frontend UI

## How to Deploy on Render.com

### Step 1 — Create GitHub Account
Go to https://github.com and sign up (free)

### Step 2 — Create New Repository
1. Click the "+" button → "New repository"
2. Name it: `myungga-management`
3. Set to **Public**
4. Click "Create repository"

### Step 3 — Upload Files
Upload all files in this folder to the repository

### Step 4 — Deploy on Render
1. Go to https://render.com and sign up (free)
2. Click "New" → "Web Service"
3. Connect your GitHub account
4. Select the `myungga-management` repository
5. Render will auto-detect settings from render.yaml
6. Click "Create Web Service"
7. Wait 2-3 minutes → Your URL will appear!

### Step 5 — Done!
Share the URL with your staff. Everyone can access from any device.

## Updating the App
1. Make changes with Claude
2. Download the new files
3. Upload to GitHub (replace old files)
4. Render automatically redeploys in 1-2 minutes
